# Registro de Estudantes 
Este projeto é uma ferramenta 
funcional para organização e
acompanhamento de progresso de 
estudantes. Ele foi desenvolvido 
com foco em **usabilidade mobile** 
e **persistência de dados local**

## Funcionalidade 
-**Organização em Tabela**
Campos para data, nome, publicação, 
lição e observação. 

-**Design Responsivo**
Visual limpo e adaptável para
telas de celulares e computadores.
